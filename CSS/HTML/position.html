디스플레이 속성

블록 속성: p태그는 기본적으로 마진값을 가지고 있다.

인라인 속성: width와 height 값을 가질 수 없다.
기본적으로 블록 속성
display의 inline-block을 사용하면
width와 height를 가지는 inline처럼 쓸 수 있다.

Visibility 속성 확인

기본 속성은 visible

hidden으로 숨겨봄

none으로 없애봄

Opacity로 속성 확인

opacity로 투명도 조절: 0에 가까울수록 투명

Position의 속성 확인

정적위치: position의 기본값
상대위치: 원래 가진 정적위치의 값에서 이동한 위치
고정위치: 브라우저의 화면 상에서 고정된 위치
절대위치: 값이 고정된 top과 left에서는 위치가 고정된 상태임 ?? 뭐야...
브라우저의 크기에 따라서 bottom과 right는 위치가 변경될 수 있음

z-index는 기본값으로 0을 갖고 있다..
